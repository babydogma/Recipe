# Portionly

Статическое приложение рецептов для GitHub Pages.

## Важно по структуре

В этой версии фотографии лежат только здесь:

```text
assets/heroes/
```

Папки `user-photos` больше нет. Старые акварельные картинки из корня `assets` удалены, чтобы не было путаницы.

## Что есть

- карточки с новыми фото;
- выбор блюда тапом по карточке;
- раскрытие ингредиентов/шагов стрелкой;
- заметки к блюдам;
- расчёт людей × дней = порции;
- пересчёт ингредиентов и КБЖУ;
- корзина выбранных блюд;
- поиск и фильтр выбранных блюд;
- сохранение выбора/заметок в LocalStorage.
