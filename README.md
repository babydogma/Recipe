# Portionly

Чистая сборка приложения рецептов.

## Структура

- `index.html`
- `style.css`
- `script.js`
- `recipes-data.js`
- `assets/` — все фотографии блюд лежат прямо здесь, без папки `heroes`.

## Важно

Для GitHub Pages просто залей содержимое этой папки в корень репозитория.
Старые папки `assets/heroes` и `assets/user-photos` можно удалить, чтобы не было каши.
