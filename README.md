# Meal Cards Planner

Статическое приложение для GitHub Pages.

## Что внутри

- категории: `Все блюда`, `Завтраки`, `Обеды / ужины`;
- твои картинки рецептов в `assets/`;
- круглый чекбокс выбора блюда поверх карточки;
- плавающая кнопка корзины снизу справа появляется только после выбора блюда;
- модальное окно корзины:
  - выбранные блюда;
  - суммарные продукты;
  - продукты `по вкусу`;
  - данные по каждому выбранному блюду;
- модальное окно блюда по нажатию на картинку:
  - большая карточка рецепта;
  - ингредиенты;
  - приготовление;
  - заметки;
- выбранные блюда сохраняются в LocalStorage;
- граммовки забиты в `recipes-data.js`, а не считываются с картинки.

## Файлы

```text
index.html
style.css
script.js
recipes-data.js
assets/
```

## Как добавить новое блюдо

1. Положить картинку в `assets/`.
2. Открыть `recipes-data.js`.
3. Добавить объект в массив `RECIPES`.

Пример:

```js
{
  id: "new-recipe",
  title: "Название блюда",
  category: "main",
  image: "assets/new-image.jpg",
  portions: 4,
  meta: "на 4 порции",
  nutrition: { kcal: 400, protein: 30, fat: 10, carbs: 50 },
  ingredients: [
    { name: "Курица", amount: 500, unit: "г" },
    { name: "Соль", amount: null, unit: "по вкусу" }
  ],
  steps: ["Шаг 1", "Шаг 2"],
  notes: ["Заметка"]
}
```

## Запуск

Открыть `index.html` в браузере.

## GitHub Pages

Закинуть файлы в корень репозитория и включить GitHub Pages:
Settings → Pages → Deploy from branch → main / root.
